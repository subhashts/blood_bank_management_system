Admin Usage Guidelines
When to Use Admin Patient Actions:
  • Assisting users who need help with the system
  • Testing system functionality
  • Emergency blood requests on behalf of hospitals
  • System demonstrations and training
  • Data verification and quality assurance
Important Considerations:
  • Always document admin actions taken
  • Respect user privacy and data protection
  • Use admin privileges responsibly
  • Verify information before making requests
  • Coordinate with hospitals and camps when needed
